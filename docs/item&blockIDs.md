# Block & Item IDs

## Summary

The Block and Item IDs must stay the same across each version as much as possible. For the ones that are not able to, however, a Migration will be necessary. But it's best to avoid that as much as possible.

So, here I will track the Block and Item IDs, and if they do not match, that will need to be fixed ASAP.

## IDs

There will be a long and short list to help make looking up quicker.

### Short (Range)

| Mod                                | IDs       |
| ---------------------------------- | --------- |
| Vanilla Minecraft ID range         | 0 - 453   |
| Doggy Talents ID range             | 454 - 468 |
| Tropicraft Block ID range          | 261 - 280 |
| Tropicraft Item ID range           | 521 - 544 |
| Equivalent Exchange Block ID range | 256 - 260 |
| Equivalent Exchange Item ID range  | 454 - 520 |

#### Notes:

Doggy Talents for 1.4.7's configs do not actually work, so the ability to set the IDs to the new IDs is non-functional.
