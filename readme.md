# Historical Mods Modpack

**Modpack is made to work with the Prism Launcher only right now.**

## Summary

Goal is to have a series of ModPacks that are all linked together through a single world. Starting from the Beta Era, up to the peak of Minecraft modding in 1.7.10. (Might consider 1.12.2, we'll see).

The world is created and played in the beta version, and should slowly be upgraded to the next version in line to continue progress through famous mods development, or demise.

## Mod Patcher

I have created a custom mod patcher that applies patches to select mods without touching the original mod source, that way we can apply needed bug fixes without messing with anyone's code.

Prism's custom command is automatically filled in to load the patcher at start-up so your computer does all the work for you.

See docs/patches.md for info on the patches.
